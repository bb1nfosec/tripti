# Getting help

Welcome! Here's where to go, depending on what you need.

| I want to… | Go to |
|---|---|
| 🌱 Make my first contribution / understand the basics | [Getting Started guide](docs/GETTING_STARTED.md) |
| 💬 Ask a question, get unstuck, or just say hi | [GitHub Discussions](https://github.com/bb1nfosec/tripti/discussions) |
| 🐛 Report a bug | [Open an issue](https://github.com/bb1nfosec/tripti/issues/new/choose) |
| 💡 Suggest a feature | [Open an issue](https://github.com/bb1nfosec/tripti/issues/new/choose) |
| 🔒 Report a security vulnerability | **Privately** — see [SECURITY.md](SECURITY.md) |
| 🧑‍💻 Understand how the code is organized | [ARCHITECTURE.md](ARCHITECTURE.md) |

**Students and newcomers especially welcome.** No question is too basic — asking is
how everyone learns. Please be kind and patient (maintainers are volunteers), and
**never paste an API key** into a question, issue, or screenshot.

By participating you agree to our [Code of Conduct](CODE_OF_CONDUCT.md).
